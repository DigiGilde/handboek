    Configureren van jouw Dev Machine

    Wij gaan ervan uit dat jouw Dev Machine een Macbook is. Deze handleiding is behoorlijk eigenzinnig, maar geeft aan welke tools een aantal van jouw collega's in de praktijk gebruiken. Voel je vrij om af te wijken, en als je denkt dat we iets moeten aanpassen in onderstaande lijst voel je vrij om hiervoor een pull request te maken.

    Zaken die standaard op je Macbook zouden moeten zitten

    • Homebrew as the missing Package Manager 
      /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

    • Keep awake met Amphetamine

    • Office DisplayLink software (verbinden externe schermen) 

      brew install --cask displaylink

    • Rectangle

      brew install --cask rectangle

    Citrix workspace

    Via Citrix krijg je toegang tot jouw digitale Rijkswerkplek.

    • Citrix workspace
    • Flex2Rijk om in te loggen en Citrix te configureren

    Communicatie

    • Mattermost voor dagelijkse communicatie met je collega's

      brew install --cask mattermost

    • WebEx voor video calls

      brew install --cask webex

    • Tuple om met elkaar vanaf afstand samen te werken

    Terminal en shell

    • Iterm2

      brew install --cask iterm2

    • Oh My Zsh

      /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/ohmyzsh/ohmyzsh/master/tools/install.sh)"

    • Autosuggestions for zsh

      git clone https://github.com/zsh-users/zsh-autosuggestions ~/.oh-my-zsh/custom/plugins/zsh-autosuggestions

    • Fish shell like syntax highlighting for Zsh

      brew install zsh-syntax-highlighting

    • Voeg plugins voor Oh My ZSH toe in ~/.zshrc. Een suggestie om mee te starten:

      plugins=(docker docker-compose git kubectl uv)

    • Touch ID in Terminal

    IDE & Coding

    We willen graag dat jij je werk goed uit kan voeren. Qua IDE maken collega's bijvoorbeeld gebruik van VSCode of JetBrains. Maar heb jij een andere voorkeur, dan kan dat waarschijnlijk ook. Mocht er een licentie nodig zijn, dan kun je dit bij jouw manager aanvragen. Voor in ieder geval JetBrains zijn er eerder licenties aangeschaft.

    Verder staan hieronder andere tools die door ons vaker worden gebruikt:

    • Sourcetree

      brew install --cask sourcetree

    • uv

      brew install uv

    • pre-commit

      brew install pre-commit

    • Xcode Command Line Tools

      xcode-select --install

    • Sign commits with SSH
